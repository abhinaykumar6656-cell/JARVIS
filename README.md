# JARVIS

An advanced cross-platform AI assistant built from scratch.

## Tech Stack

- Python (FastAPI)
- React Native
- Electron
- SQLite (Development)
- PostgreSQL (Production)
- Ollama
- WebSocket

## Features

- Voice Assistant
- AI Agents
- Memory
- Automation
- Mobile + Desktop Ecosystem
- Wake Word Detection
- Sleep Mode
- Real-time Synchronization
